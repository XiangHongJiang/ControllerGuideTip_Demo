# ControllerGuideTip_Demo

#### 项目介绍
给界面添加操作引导，适用场景：

1. App首次启动，进入某些界面，需要指导用户使用。
2. App点击某项功能，需要引导用户操作。
3. 需要在任意界面，自定义任意操作引导的情况。
